ESP32 Web Flasher

BWifiKill ESP32 V4.0

Instala el firmware completo directamente desde el navegador. Usa Chrome o Edge, conecta el ESP32 por USB y selecciona el puerto serial cuando el navegador lo pida.

1. Conecta

Usa un cable USB de datos y cierra cualquier monitor serial abierto.

2. Instala

Presiona el boton y elige tu ESP32 en la ventana del navegador.

3. Reinicia

Al terminar, reinicia la placa y espera el splash de BWifiKill.

Release local V4.0

Instalador

El manifest usa el binario completo ubicado en binarios/BWifiKill-ESP32-V4.0-full.bin.

Si el puerto no aparece o la escritura falla, manten presionado BOOT mientras inicia el flasheo. Usa esta herramienta solo en hardware propio.
